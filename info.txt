==== Star Citizen Guild Bot ====

Features:   
    - Dedicated organisation trusted members
        - Add specified users to DB
    - Credit Splitter
        - Add usernames, Add amount of money made, Divide money made by amount of users - Output embed with the value of money to be sent to each person.
        - Tag Dedicated user
        - Tag seller
    - Event Organiser
        - Event signup, Time and date, countdown, output.
    - Star Citizen API Integration
        - users
        - Orgs
        - updates for PTU
        - ships
